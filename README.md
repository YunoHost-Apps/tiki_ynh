## Overview

Tiki Wiki CMS Groupware is the Free / Libre / Open Source Web Application with the most built-in features.

### Use cases
- Web Publishing
- Collaboration
- Project Management
- Office Suite
- Knowledge base
- Shopping Cart
- Social Networking
- CRM
- Membership
- E-learning
- Tiki Trackers is the built-in database web apps builder and low-code / no-code application framework.
